# Assets

The Würth Rotación dashboard ships **no image assets** — no logo files, no icons, no
illustrations, no photography. This is accurate to the source repository
(`elterco2012-dev/alerta-de-fluctuacion`), which is a Streamlit app styled entirely
with CSS-in-Python.

## Brand mark
The product brands itself with a **text wordmark** only:

> **Würth Argentina** — *Alertas de Rotación*

rendered at `800/20px` in `#1a1a2e`, optionally prefixed with the 🔔 emoji
(used as the page icon / favicon in the source). The official Würth corporate
logomark is **not** reproduced — use the text treatment shown in the UI kit.

## Iconography
Emoji + Unicode glyphs, typed directly (no files). See the **ICONOGRAPHY** section
of `../README.md` for the full stable emoji→meaning mapping.

If a future surface genuinely needs raster/vector assets, add them here and document
them in the README index.
